# Employee-Mental-Wellbeing-Insights-using-Machine-Learning
The Employee Wellbeing Insight Tool is a machine learning–based web application designed to provide supportive workplace wellbeing insights.
Users answer a short, anonymous questionnaire related to personal, workplace, and communication factors. Based on the responses, the system offers positive, non-diagnostic insights that encourage awareness of wellbeing resources and healthy work habits.

This tool is intended only for general guidance and awareness and does not replace professional advice.

## Key Features:
- Clean and user-friendly multi-page web interface

- Machine learning model trained on survey data

- Predicts likelihood of seeking wellbeing support

- Provides friendly, non-judgmental insights

- Fully anonymous — no data is stored

- Deployed as a web application using Flask and Render

## Tech Stack Used:
- Programming Language: Python

- Machine Learning: Scikit-learn

- Data Processing: Pandas, NumPy

- Web Framework: Flask

- Frontend: HTML, CSS

- Model Serialization: Joblib

- Deployment Platform: Render
